Instrucciones de instalacion

1. Importe db_videos.sql en su base de datos.
2. Actualizar archivo config.php
3. Ejecute el archivo index.php para cargar archivos de vídeo.
4. Ejecute el archivo ver_videos.php para ver los archivos de vídeo cargados.
